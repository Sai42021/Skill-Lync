# Skill-Lync
Website that enables clients to contact free lance laborers and tutors. This is a prototype... Development on a final site is underway.

## Link to Home page
https://sai42021.github.io/Skill-Lync/Home.html

## Mission
We seek to create a platform that not only provides income opportunities,
to unemployed youth and freelance agents but also make it easy to reach,
the right talent and skill nationwide.

## Goals
* Reach worldwide markets.
* Use data and machine learning algorithms to identify skill gaps and market trends.
* Implement a user-friendly interface.
* Leverage emerging technologies.

## Solutions
* Offer skilled unemployed youth and freelance agents an opportunity to make an income
* Make it 100% frictionless to access skills and talent nationwide.
